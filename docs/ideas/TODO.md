# Feature Roadmap

Features to extend the UI so Remes feels like a complete sales operating system.

## High Priority

- [ ] **Email Signup Not Working** — Not getting verification emails from supabase. Need to investigate the root cause and if there are any settings we need to flip.
- [ ] **Persistent Contact Book** — `/contacts` page aggregating all enriched contacts across sessions. Searchable, filterable, shows enrichment status, email history, originating session.
- [ ] **Company Watchlist** — Bookmark companies to a persistent `/pipeline` view. Group by stage (Researched / Contacted / Replied). Kanban or table with stage badges.

## Medium Priority

- [ ] **Signal Feed / Activity Timeline** — Chronological feed of all platform activity: research runs, enrichments, emails sent. Makes the app feel alive between research sessions.
- [ ] **Outreach Templates & Playbooks** — Save winning email patterns as reusable templates. Bundle ICP + outreach style into one-click playbooks.
- [ ] **Expanded Settings Page** — Real multi-tab `/settings` page: Profile, Connections, Email Preferences, Templates, Usage (credits/quotas).

## Lower Priority

- [ ] **Analytics / Activity Reports** — Emails sent over time, signal types detected, companies researched by week. First-party data only (no open/click tracking until CASA clears).

## Out of Scope (For Now)

- Email open/click tracking (blocked on Google CASA)
- Team collaboration features (near-term roadmap, not this phase)
- CRM integrations (Salesforce, HubSpot sync)
