# Signal

AI-powered outbound sales that finds high-intent leads without the cost of a sales hire.

Signal monitors the web for buying signals (job postings, funding rounds, hiring patterns, tech stack changes), identifies companies experiencing a pain you solve, finds the right contact, and drafts a personalized email rooted in the signal it detected.

## Stack

Next.js, TypeScript, Tailwind, shadcn/ui, Anthropic API, Parallel.ai

## Getting started

```bash
npm install
npm run dev
```

Open http://localhost:3000.
